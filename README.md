## ColourConverter
It's a pretty basic colour converter.. made in glfw and imgui

Currently it supports the following formats ->
- RGB
- HEX
- RGBA
- RGBA (Floating Points)
- HSV <br>
> More formats would be added in the future

**It also contains a colour picker!**
